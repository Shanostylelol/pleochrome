## What's ACTUALLY Missing (After Corrections)

### CRITICAL: Type-Specific Behavior
- Subtask types change a dot color but trigger NO type-specific UI
- Document type should show upload zone inline
- Meeting type should show meeting scheduler
- Approval type should show approval flow
- Call/Email/Note types should show appropriate log forms
- This is the #1 gap per Shane's feedback

### CRITICAL: Universal Conversation System
- Comments exist at asset and task level but NOT at subtask or stage level
- Notes field on subtasks has no save button, no timestamp, no author
- Need unified conversation/activity thread at EVERY entity level
- @mentions work in router but notification creation is inconsistent

### CRITICAL: File Attachment at All Levels
- Files only attachable to tasks (TaskFileList)
- Need subtask-level and stage-level file attachment
- Need document version history (parent_document_id exists in schema, not wired)
- Need notes/conversation on documents

### HIGH: Notification Triggers Incomplete
Most notification types aren't being created:
- ❌ comment_reply notifications
- ❌ subtask_assigned notifications
- ❌ approval_decision notifications
- ❌ stage_completed notifications
- ❌ phase_advanced notifications
- ❌ document_uploaded notifications
- ❌ deadline_approaching (needs cron)
- ❌ deadline_overdue (needs cron)

### HIGH: Approval Chain Not Auto-Advancing
- Approval request creates records
- But task doesn't auto-advance when all approvers approve
- Sequential vs parallel approval logic not implemented
- Approvals page shows empty because getMyQueue doesn't aggregate correctly

### HIGH: Template Editor Incomplete
- Can add stages/tasks but can't DELETE them
- Can't reorder via DnD
- templates.removeStage/removeTask/removeSubtask procedures missing

### MEDIUM: Missing Features
- Required documents checklist (which docs are needed vs uploaded per stage)
- Batch document download (ZIP)
- Report generation (generate_asset_report RPC exists but no UI)
- Partner onboarding/credentials tabs (components not built)
- Ownership tree visualization (router exists, no UI)
- Communication logging on contacts/partners (CommunicationsTab built but only on asset detail)
- Reminders have no dashboard widget or getDue() query
- Activity export (CSV download button exists but export query may not work)
- Task calendar view
