## Key Findings
1. ERC-3643 (T-REX) is the clear winner for security tokens — $32B+ tokenized, DTCC joined
2. SPV (LLC) is standard legal structure — asset owner → SPV holds asset → SPV issues tokens
3. Chainlink Secure Mint = oracle-gated minting that prevents unbacked token creation
4. Tiamonds = 1:1 NFT-backed diamonds, GIA certified, Lloyd's insured, Liechtenstein vault
5. NYBlue ZIRC = 1:1 fungible token backed by blue zircon, Imperial Vaults custody
6. PAXG = LBMA gold bars, Brink's vaults London, OCC-regulated, monthly audits
7. tZERO = SEC/FINRA-regulated ATS, 24/7 order entry, CLOB + auction + block trading
8. Typical tokenization costs: $50K-$3M+ depending on complexity
9. Money transmitter: securities BD exemption exists but functional analysis still applies
10. SEC/CFTC MOU (Mar 2026) = historic safe harbor with "Transition Point" mechanism
